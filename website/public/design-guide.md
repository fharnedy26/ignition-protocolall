# Ignition Protocol - Design Guide

## Visual Flow: Poster → Website → Book Cover

This guide describes the visual design system that creates a seamless flow from the exhibition poster, through the website, to the project book cover.

## Design Philosophy

**Flame & Darkness Theme**: A sophisticated dark aesthetic with flame-inspired accents that represents the combustion science at the heart of the project.

## Color Palette

### Primary Colors
- **Deep Black**: `#0a0a0a` - Primary background
- **Dark Card**: `#1a1a1a` - Secondary surfaces
- **Off-White**: `#f5f5f5` - Primary text

### Flame Accents
- **Yellow**: `#ffd700` - Brightest flame (highlights)
- **Amber**: `#ffb347` - Warm glow
- **Orange**: `#ff6b35` - Primary accent
- **Red-Orange**: `#ff4500` - Deep flame
- **Deep Red**: `#8b0000` - Ember glow

## Typography

### Poster
- **Title**: 180px, Bold, Uppercase
- **Subtitle**: 48px, Light
- **Body**: 32px, Regular

### Website
- **Hero Title**: 6xl-8xl (96-128px), Bold
- **Section Headings**: 3xl-5xl (30-48px), Bold
- **Body**: Base-lg (16-18px), Regular

### Book Cover
- **Title**: 72px, Bold
- **Subtitle**: 18px, Light, Uppercase
- **Body**: 10-12px, Regular

## Visual Hierarchy

### Poster (Eye-Catching, Bold)
1. **Large Title** - Dominates top third
2. **Visual Element** - Central molecular visualization
3. **Key Stats** - Bottom left
4. **Exhibition Info** - Bottom right

### Website (Interactive, Informative)
1. **Hero Section** - Full viewport, gradient background
2. **Content Sections** - Scrollable, organized
3. **Navigation** - Always accessible
4. **Footer** - Exhibition info

### Book Cover (Refined, Professional)
1. **Title** - Top third, elegant
2. **Molecular Icon** - Center, subtle
3. **Exhibition Info** - Bottom, minimal
4. **Decorative Corners** - Subtle framing

## Design Elements

### Gradients
- **Flame Gradient**: Yellow → Orange → Red-Orange
- **Background**: Deep Red → Black
- **Radial Glows**: Flame colors with low opacity

### Animations
- **Flame Flicker**: Subtle brightness variation
- **Glow Pulse**: Expanding shadow effects
- **Ember Glow**: Deep red pulsing

### Molecular Visualizations
- **Poster**: Large, animated circles (representing molecules)
- **Website**: Charts and data visualizations
- **Book Cover**: Refined molecular structure icon

## Usage Instructions

### Poster
1. Open `poster.html` in a browser
2. Print at 24" × 36" (or scale as needed)
3. Add QR code linking to website
4. Use for exhibition display

### Book Cover
1. Open `book-cover.html` in a browser
2. Print at 8.5" × 11" (standard book size)
3. Use as front cover of project book
4. Ensure high-quality printing for best results

### Website
- Already implemented in Next.js
- Uses same color palette and design language
- Responsive across all devices

## Exporting as Images

To export these designs as PNG/PDF:

1. **Using Browser**:
   - Open HTML file
   - Use browser's print function
   - Save as PDF
   - Convert PDF to PNG if needed

2. **Using Screenshot Tool**:
   - Open HTML file
   - Use full-page screenshot extension
   - Save at high resolution (300 DPI for print)

3. **Using Design Software**:
   - Open HTML in browser
   - Take screenshot
   - Import into Photoshop/Illustrator
   - Refine and export

## Print Specifications

### Poster
- **Size**: 24" × 36" (A1 or custom)
- **Resolution**: 300 DPI minimum
- **Color**: CMYK for professional printing
- **Paper**: Matte or semi-gloss

### Book Cover
- **Size**: 8.5" × 11" (Letter)
- **Resolution**: 300 DPI
- **Color**: CMYK
- **Paper**: Cover stock, matte finish

## Brand Consistency

All three designs share:
- Same color palette
- Same typography hierarchy
- Same flame & darkness aesthetic
- Same exhibition information
- Same author attribution

This creates a cohesive visual experience that guides viewers from the poster → website → book.
